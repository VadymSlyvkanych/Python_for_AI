Домашнее задание 8
Идеи для мультимодальных AI-приложений

Мультимодальность — это комбинирование разных типов данных
(текст, аудио, изображения, видео) для решения задач, которые
недоступны каждой модальности по отдельности.

1. AI-репетитор по видео-лекциям

   Модальности: аудио + видео (изображения) + текст
   Стек: Whisper (речь в текст) + Gemini Vision (анализ
   слайдов/доски) + LLM (генерация конспекта)

   Как работает:
   - Пользователь даёт ссылку на YouTube-лекцию
   - Whisper распознаёт речь лектора в текст с таймкодами
   - Gemini Vision анализирует каждый слайд/изображение с экрана
   - LLM объединяет текст лекции + содержимое слайдов в
     структурированный конспект
   - Пользователь может задавать вопросы по материалу, и система
     находит нужный фрагмент видео + слайд

   Пример: видео-лекция по нейросетям → конспект с формулами
   со слайдов, определением терминов и ссылками на таймкоды

2. AI-модератор встреч и совещаний

   Модальности: аудио + видео (эмоции/жесты) + текст
   Стек: Whisper (речь) + facial expression analysis (эмоции) +
   LLM (генерация протокола)

   Как работает:
   - Whisper распознаёт речь всех участников с диарной сегментацией
     (кто когда говорил)
   - Видео-анализ определяет эмоции (согласие, несогласие,
     замешательство) по выражению лиц
   - LLM собирает стенограмму с пометками:
     * "все согласны" (after nod detection)
     * "спорный момент" (после обнаружения несогласия)
     * "задача" (когда прозвучало поручение)
   - На выходе: протокол с таймкодами, задачами и решениями

3. AI-помощник для локализации контента

   Модальности: аудио + текст + аудио (TTS)
   Стек: Whisper (распознавание) + LLM (перевод) +
   TTS (озвучка)

   Как работает:
   - Пользователь загружает видео без субтитров на языке X
   - Whisper распознаёт оригинальную речь
   - LLM переводит текст на нужный язык с сохранением стиля
     и таймингов
   - TTS (например, Kokoro или ElevenLabs) озвучивает
     переведённый текст
   - На выходе: видео с новой аудиодорожкой на выбранном языке

   Применение: лекции, интервью, вебинары — мгновенная
   локализация без участия человека

4. Ассистент для первичной диагностики

   Модальности: аудио (диалог) + изображения (снимки) + текст
   Стек: Whisper (расшифровка диалога) + vision model
   (анализ снимков МРТ/рентген) + LLM (формирование отчёта)

   Как работает:
   - Записывается разговор врача и пациента
   - Whisper расшифровывает диалог (симптомы, жалобы, история
     болезни)
   - Vision model анализирует прикреплённые снимки (МРТ,
     рентген, КТ)
   - LLM сопоставляет симптомы + результаты анализов и
     формирует предварительный отчёт для врача

   Важно: финальное решение всегда остаётся за врачом, AI
   только помогает не упустить детали
